# TAS-Editor-Online

A simple-to-use GUI for Nintendo Switch TAS editing, on the web.

# Fork
Note that this is a web fork of the original [TAS-Editor](https://github.com/Jadefalke2/TAS-Editor/) by Jadefalke2, MonsterDruide1, and others.
